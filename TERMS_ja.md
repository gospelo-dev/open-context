本規約は `https://open-context.gospelo.dev` の**ホスト版サービス**を対象とします。**ソースコードには適用されません** — コードは [MIT ライセンス](LICENSE)で提供され、自前運用が可能です。以下のいずれかがあなたの用途に受け入れられない場合は、**自社運用してください**([自社運用](#自社運用)参照)。

## 1. 現状有姿・無保証・SLA なし

ホスト版は**ベストエフォートの「現状のまま(as-is)」提供で、いかなる保証も SLA もありません**。可用性・性能・正確性は保証されません。予告なく変更・レート制限・提供停止することがあります。

## 2. あなたの GitHub トークンとデータ

- あなたは**自分の GitHub トークン**(BYO)を渡します。これはパッケージの docs・ソース取得のため、**あなたの代わりに** GitHub API を呼ぶ目的にのみ使用します。
- **トークンは保存しません。** BYO 方式ではトークンを一度検証し、**SHA-256 ハッシュ → 識別情報**のみを約1時間キャッシュします。生のトークンは永続化しません。(任意の OAuth 方式ではトークンを AES-GCM 暗号化して保存。既定では無効。)
- あなたのトークン・識別情報を第三者に**販売・共有しません**。
- リクエストは性質上 **GitHub**(api.github.com / raw / codeload)および **Cloudflare**(ホスティング基盤)を経由します。当該通信にはそれぞれの規約・プライバシーポリシーが適用されます。
- セキュリティモデルと脆弱性報告は [SECURITY.md](SECURITY.md) を参照。

## 3. 許容される利用

- リクエストは**あなた自身の** GitHub トークン・レート制限・権限で実行されます。トークンに付与するスコープはあなたの責任です(必要最小限に)。
- サービスの濫用・過負荷・妨害を試みないこと。権限のないコンテンツへのアクセスに利用しないこと。
- **GitHub の利用規約および API レート制限**を尊重すること。
- 取得・返却されるドキュメント/ソースコードは**各リポジトリの著作権者に帰属し、それぞれのライセンスに従います**。本サービスは取得の導管にすぎず、当該コンテンツの権利を付与しません。利用者は各プロジェクトのライセンスに従って利用する責任を負います。

## 4. 自社運用

トークンやリクエストをホスト版経由で流すことが組織的に受け入れられない場合(社内・機密・コンプライアンス要件など)は、**自前でインスタンスを運用してください**。コードは OSS(MIT)です — [デプロイ手順](docs/current/architecture/deployment.md)を参照。こうしたケースでは自社運用を推奨します。トークンと通信をすべて自社インフラ内に留められます。

## 5. 責任の制限

法律が許す最大限の範囲で、メンテナはホスト版サービスの利用(または利用不能)から生じるいかなる直接・間接・付随的・結果的損害についても**責任を負いません**。これはサービス文脈における [MIT ライセンス](LICENSE)の無保証条項を補強するものです。

本規約は随時更新されることがあり、最新版は本リポジトリにあります。変更後もホスト版の利用を継続した場合、変更に同意したものとみなします。質問・問題は[プロジェクトリポジトリ](https://github.com/gospelo-dev/open-context)へ。
